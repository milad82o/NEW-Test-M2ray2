Step-by-Step Guide

Step 1: Generate Files

Go to the Generator tab, click "Generate Files", then "Download All 6 Files".

Step 2: Fork a Repository

Go to github.com/amircloner/g2ray and click Fork.

Give your fork a name, check "Copy the main branch only", click Create fork.

Step 3: Upload Files to .devcontainer

In your fork, navigate to .devcontainer folder:

yourname/g2ray — .devcontainer
📁 .devcontainer
  📄 config.json — replace
  📜 setup.sh
  🐋 Dockerfile
  📄 devcontainer.json — replace
  📄 show-link.sh — replace
  📜 start.sh

For each marked file:

1Click the filename
2Click the pencil icon (Edit this file) in the top-right
3Select all (Ctrl+A) and delete
4Paste the content from your downloaded file
5Commit changes at the bottom
Important: Dockerfile must have no file extension (not .txt or .sh). GitHub editor handles this correctly by default.

Step 4: Create Codespace

In your fork, click the green Code button → Codespaces tab → + New codespace on main.

github.com/yourname/g2ray
Code
Codespaces
No codespaces yet
+ New codespace on main

Wait 3-5 minutes for build.

Step 5: Get Your VLESS Link

After the codespace builds, the terminal auto-runs start.sh which shows your link:

Terminal
# Xray installed, starting...
[g2ray] Starting Xray in tmux...
[g2ray] Downloading Xray v26.3.27...
[g2ray] Done.

vless://xxxx-xxxx@94.130.50.12:443?...

If the link is not shown, run in terminal:

Terminal
$ show-link.sh
Clients: Android: v2rayNG / Nekobox  |  Windows: v2rayN / Nekoray  |  iOS: Streisand / FoXray  |  Linux: Nekoray

Troubleshooting

⚡
Xray not running: Run bash /usr/local/bin/start.sh in terminal
⚡
Idle timeout: Set github.com/settings/codespaces → Default idle timeout → 240 minutes
⚡
DNS error: The setup.sh downloads GeoIP/GeoSite automatically. If missing: curl -sL "https://github.com/v2fly/geoip/releases/latest/download/geoip.dat" -o /usr/local/bin/geoip.dat
⚡
Connection timeout: Try different transports (ws, h2, grpc) via config.json edit
⚡
Free quota: 120 core-hours/month = ~60 hours real time on 2-core codespace. Stop when done.
